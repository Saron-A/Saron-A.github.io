ABOUT ME

Hi, I’m Saron Abebe — a front-end developer and Software Engineering student at AASTU. I specialize in HTML, CSS, JavaScript, and React, with a focus on building responsive, accessible, and user-friendly interfaces.

Outside the classroom, I’ve been expanding into full-stack and mobile development through The Odin Project and hands-on experience with React Native and Flutter. I also hold certifications from SheCodes, which strengthened my skills in modern web tools and best practices. My projects include a React shopping cart, a book downloader using the Google Books API, a responsive daycare website, a personal poetry blog, a Flutter habit tracker, and most recently, TodoDaily — a React Native task manager with persistent local storage. Each one reflects my goal to build purposeful, user-centered digital experiences.

I’m passionate about creating tools that blend creativity and real-world impact, and I’m currently seeking opportunities to grow with like-minded development teams.

PROJECTS

ToDo Daily mobile application

A minimalist mobile task manager built with React Native and Expo. Users can add, complete, and delete daily tasks with a smooth and intuitive UI. Data is stored locally using AsyncStorage to ensure persistence across sessions.

Tech Stack: React Native, Expo, JavaScript, AsyncStorage, React Hooks, Flexbox

Mini Shopping Cart

A dynamic React-based shopping cart application that fetches products from an external fake-data API and allows users to browse, adjust quantities, add items to cart, and proceed to checkout. The app demonstrates client-side routing, state lifting, and responsive UI design.

Tech Stack: React, React Router Dom, CSS, Axios and Netlify

Poetry Blog

A minimalist, responsive poetry archive showcasing my original poems. Inspired by classic literary aesthetics, it features genre filtering, an author bio, and a clean layout for an immersive, emotionally engaging reading experience.

Tech Stack: HTML, CSS, JavaScript, Firebase, API Integration, mySQL,Version Control

Book Downloader Web App

A sleek and functional web application that allows users to search for books using the Google Books API. It features detailed book previews, downloadable formats when available (EPUB/PDF), filtering options, and a dynamic UI for seamless browsing.

Tech Stack: HTML and CSS, JavaScript, API integration and React

Ankilba Daycare Website

Ankelba Daycare Website is a responsive, culturally-inspired platform that blends Ethiopian heritage with modern design. It highlights the daycare’s values, services, and mission through visual storytelling and includes a contact form for inquiries and enrollment

Tech Stack: HTML, CSS, JavaScript and FormSpree
