## Introduction

This HTML document serves as the main structure for Saron's personal website. It includes sections such as an introduction, education, skills, projects, and a contact form.

## Technologies Used

The following technologies were used to build this website:
- HTML5
- CSS3
- JavaScript
- Bootstrap
- jQuery
- Font Awesome
- Google Fonts

## Features

- Responsive design for various screen sizes.
- Navigation menu for easy access to different sections.
- Sections for introduction, education, skills, projects, and contact information.
- Timeline feature to showcase education and project experiences.

## Installation

To view the website locally, simply clone this repository to your local machine and open the `index.html` file in a web browser.

```bash
git clone https://github.com/Saron-A/saron-portfolio.git
```

## Usage

Once the repository is cloned, open the `index.html` file in any web browser to view Saron's portfolio website. Navigate through the different sections using the navigation menu.
